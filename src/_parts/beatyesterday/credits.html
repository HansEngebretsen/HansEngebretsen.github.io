Details & Tech

The client on this project was a backend developer, and was building out the functionality of the app. Because of this, there was no need for any type of CMS, or complicated deliverables, and the full source (Sass, and HTML partials) were able to be levereaged for maitenence and integration. I used git for version control and to collaborate throughout the project with the client. To keep things private, I hosted the remote on Dropbox. For the build process, and concating the html/js partials, I used my typical workflow of Grunt/Sass etc.
