Details & Tech

Nerdy Stuff

The interactive experience was built on the Laravel php framework, and an an API built (by Dustin Hayes) on top of Garmin Connect. I built the style guide using a custom jekyll architecture simular to the one found here to be able to use patials, but still generate static files. The email was built using a custom php script, and gif generator, check out the source.

The People

From a creative standpoint, I collaborated throughout the project with Garmin creative directors Ryan Glendening and Matt Bowne. We had great support from the Garmin Connect developer team, as well as Dustin Hayes who was the lead developer on letsbeatyesterday.com, and helped with the email as well.
