Details & Tech

The client for JournalBot was a backend developer, and was building out the functionality of the app, so there was no need a CMS, or complicated deliverables, and the full source (Sass, and HTML partials) were able to be levereaged for integration/maitenence. I used git for version control and to collaborate throughout the project with the client. To keep things private, I hosted the remote on Dropbox. For the build process, and concating the html/js partials, I used my typical workflow of Grunt/Sass etc.
